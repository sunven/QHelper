QHelper 前端助手 - 隐私政策

生效日期: 2025 年 1 月 | 版本: 1.0

概述

QHelper 前端助手（以下简称"本扩展"）是一款为前端开发者设计的浏览器实用工具扩展。我们高度重视您的隐私，并承诺保护您的个人信息。

数据收集

本扩展不收集任何用户数据

  • 不收集用户浏览历史
  • 不收集用户输入的内容
  • 不向任何服务器发送数据
  • 不使用任何分析工具或追踪服务
  • 不与第三方共享任何信息

权限说明

本扩展请求以下权限，仅用于实现特定功能：

1. cookies 权限

用途: "清除 Cookie" 功能

  • 仅在您主动点击"清除 Cookie"按钮时使用
  • 用于读取和删除当前网站的 Cookie
  • Cookie 数据不会离开您的浏览器
  • 不会记录或传输任何 Cookie 内容

2. tabs 权限

用途: 在新标签页中打开工具

  • 仅用于创建新标签页显示各种工具界面
  • 不读取标签页内容或浏览历史
  • 不追踪您访问的网站

3. storage 权限

用途: 存储用户偏好设置

  • 仅存储您最近使用的工具记录
  • 所有数据保存在浏览器本地存储中
  • 数据仅由您本人访问

4. host_permissions（<all_urls>）

用途: GitHub 页面集成功能

  • 仅在 GitHub 仓库页面注入内容脚本
  • 功能：添加一个"在 vscode.dev 中打开"的按钮
  • 不读取、修改或传输任何页面内容
  • 不在其他网站运行任何脚本

数据存储

所有数据均存储在您的浏览器本地：

  • Cookie 数据：仅存储在浏览器 Cookie 存储区
  • 用户偏好：仅存储在浏览器本地存储（chrome.storage.local）
  • 不会同步到云端服务器
  • 不会在设备间同步

第三方服务

本扩展：

  • 不使用任何第三方服务
  • 不嵌入任何第三方代码
  • 不连接任何外部 API
  • 不显示任何广告

数据安全

  • 所有数据处理均在您的浏览器本地完成
  • 无服务器交互，无数据传输风险
  • 无数据泄露可能性

儿童隐私

本扩展面向成年开发者使用。我们不有意收集 13 岁以下儿童的任何信息。

隐私政策变更

如果我们更新本隐私政策，将在扩展更新说明中通知用户。继续使用扩展即表示您接受更新后的政策。
